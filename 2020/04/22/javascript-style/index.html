在JavaScript中可以对html中的元素进行属性和样式的修改

首先对html中的元素进行查找，查找html标签有以下几种方法：

  • 通过 id 查找 HTML 元素
    document.getElementById()，参数填入id值即可返回id一样的元素。
  • 通过标签名查找 HTML 元素
    document.getElementsByTagName()，参数填入标签即可返回所有该标签。
  • 通过类名查找 HTML 元素
    document.getElementsByClassName()，参数填入类名即可返回所有相同类名的元素。

然后即可对元素的属性和样式进行修改：

  • 改变属性的值
    使用查找返回的元素对象.属性值=” “；进行修改。
  • 改变元素样式
    使用查找返回的元素对象.style.CSS属性=” “；进行修改。

onmouseover事件：鼠标移动都某个指点的HTML标签上，执行的事件。

onmouseout事件：鼠标移出某个指点的HTML标签后，执行的事件。

示例：

![在这里插入图片描述](https://img-blog.csdnimg.cn/20200420191659923.gif =300x250)
我们可以使用这结合两个事件实现一些页面交互效果，例如上图，当鼠标移动到微信图标上的时候显示二维码，移出时二维码消失，显示微信图标。

实现代码：

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<script>
			//当鼠标移入执行此方法
			function onover(){
				var msg=document.getElementById("msg").innerHTML="作者微信二维码";
				var wechat=document.getElementById("wechat");
				wechat.src="img/Wechat.jpg";
				wechat.style.width="200px";
				wechat.style.height="200px";
			}			
			//当鼠标移出执行此方法	
			function onout(){
				var msg=document.getElementById("msg").innerHTML="";
				var wechat=document.getElementById("wechat");
				wechat.src="img/icon.png";
				wechat.style.width="30px";
				wechat.style.height="30px";
			}
		</script>
	</head>
	<body>
		<div align="center">
			<img id="wechat" src="img/icon.png" width="30" height="30" onmouseover="onover()" onmouseout="onout()" />
			<p id="msg"></p>
		</div>
	</body>
</html>
